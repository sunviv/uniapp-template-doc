开发指南
示例页面

项目规范

集成 ESLint、StyleLint、EditorConfig 代码格式规范；集成 git hooks 自动校验 git 提交代码格式。

主题、样式

集成原子化 CSS，简化样式编写，提升开发效率；统一组件库，支持主题定制。

Vue3、TypeScript

采用 Vue3、TypeScript 开发；支持组件自动加载、API 自动导入。

开发环境优化

使用 VSCode 开发，提升开发效率；支持多环境打包构建。

权限控制

添加页面跳转拦截，登录权限校验。

状态管理

集成 Pinia 状态管理，支持模块化开发。
